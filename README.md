# Serve current folder with SFTP

```bash
# for macOS
curl -L https://github.com/yene/sftpserver/releases/download/0.1.0/sftpserver-macos --output sftpserver
chmod +x ./sftpserver
./sftpserver
# login with any user, and the given password
```
